DOM events are action that occur as a result of the user action or as result of state change of the elements of a DOM tree.
    Examples of HTML events:

  • when a user clicks the mouse
  • when a web page has loaded
  • when an image as been loaded
  • when the mouse moves over an element
  • When an input field is changed
  • When an HTML form is submitted
  • When a user stokes a key
